### 项目目录

```
|- libs    // 第三方库
|
|- configs // 场景中物体的配置
|
|- public  // 公共资源
|
|- src     // 源码
|
|- game.js  // 小游戏入口
|
|- game.json // 小游戏配置
|
|- project.config.json // 项目配置
|
|- README.md   // 项目说明
```

### 项目特点

- 项目使用ES6 + Three.js开发
- 代码规范采用standard规范
- 游戏逻辑使用面向对象进行模块化开发
- 使用Gulp作为项目工程化工具
- 通用功能的封装，比如：动画
- 游戏整体采用MVC的架构模式进行管理和控制
- 基本实现微信跳一跳小游戏的玩法
